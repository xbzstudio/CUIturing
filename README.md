### 一、登录您的账号

在程序启动时，您将会看到程序输出"cookie:"，这代表您需要输入您的cookie，获取cookie的方法如下：

请您打开 https://icodeshequ.youdao.com/ ,按下F12后，选择"Network"或"网络"选项卡,找到名为"/"的文件,查看请求头,找到"cookie"项,粘贴至程序

当程序输出"小图灵控制台版登陆成功，请输入功能编号"时，本程序初始化成功。
